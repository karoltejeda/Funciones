Etiquetas Extras

Etiqueta pre

La etiqueta pre sirve para respetar lineas y espacios.

Hola soy Karol tengo 16 años 

Hola      soy 
    Karol       tengo       16 años.

Etiqueta abr

La etiqueta abr se utiliza para las abreviaturas.

Segun datos de la OMS Consumo de exceso de azucar provoca daños severos en la salud de los niños.

Etiqueta bdo

Si se desea cambiar la orientacion de un texto, se utiliza la etiqueta bdo.

Este es un texto normal

Este es un texto diferente

Etiqueta cite

La etiqueta Cite se utiliza para describir citas textuales.

of course my horseee por: Karol Pamela

Etiqueta small

La etiqueta small hace el texto mas pequeño.

Este texto esta dentro de una etiqueta small

Etiqueta sub

Si se desea escribir un subíndice se debe utilizar la etiqueta sub.

298 5

Etiqueta sup

Si se desea escribir un superindice, se debe utilizar la etiqueta sup.

298 5

Etiqueta time

Para escribir fechas y horas se debe utilizar la etiqueta time.

Mañana es dia 2024-11-06

Mañana es dia 6 de noviembre de 2024

Etiqueta adress

Para especificar al navegador que el texto se refiere a una dirección, utiliza la etiqueta adress.

Loma dorada No. 100 Colonia lomas de rosales CP 89100 Tampico, Tamps.
